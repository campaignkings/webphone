Thank you for choosing Campaign Kings.

In order to use Campaign Kings WebPhone, upload the code inside this folder into a subdirectory from your Vicidial AGC directory (typically /srv/www/htdocs/agc/webphone), then point your webphone URL to the absolute or relative path within your server, such as /agc/webphone/ckphone.php.php.

If you are using Campaign Kings WebPhone in conjunction with Vicidial you will need to change this setting in ADMIN → System Settings.
